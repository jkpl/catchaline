# catchaline

TCP server for catching arbitrary lines of text.

## Usage

    -p, --port NUMBER  9999            Listen port
    -h, --host HOST    localhost       Bind to host
    -o, --output FILE  catchaline.txt  Output file
        --help                         Print this help
